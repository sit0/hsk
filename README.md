# hskb.py

`hskb.py` is a little Python 2 script providing key bindings for most common 
actions in Hearthstone under Linux.

# Installation

Make sure you have python-xlib and PyMouse. In Ubuntu:

```
# apt-get install python-xlib python-pip
# pip install pymouse
```

# Usage

`$ python hskb.py`

# Have fun!
